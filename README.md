# dev-2026-grading

这是 `dev-2026` 系列考核题的提交与评分仓库。

## 学生如何提交

请不要在这里直接上传代码。

你需要做的是：

1. 先完成你自己的题目仓库
2. 不要修改题目仓库名
3. 回到本仓库，点击 `Issues`
4. 使用提供的提交模板新建 issue
5. 填写：
   - 你的 GitHub 用户名
   - 题目仓库名
   - 你的提交仓库地址
   - 备注（如果有未完成部分，可以说明）

## 提交时请注意

- 题目仓库名不要改
- 提交仓库地址要能访问
- 如果有没做完的部分，可以在备注里提前说明

## 当前支持的题目仓库名

- `dev-2026-01`
- `dev-2026-02`
- `dev-2026-03`
- `dev-2026-04`
- `dev-2026-05`
