# 面向 agent 的说明

## 这不是学生文档

本文件只给评分 agent / 操作者看。

## 评分原则

- 优先按题目目标评分
- 面向新人，不按资深工程师标准打分
- 缺失部分可以记为 partial，不要默认直接作废
- issue 一定要回复
- CSV 尽量一定要写
- 最后关闭 issue
