# Rubric

Total score: 100

## 1. Environment setup and runnability, 20

Check:
- Whether dependencies can be installed
- Whether the program can start correctly

Suggested scoring:
- 16-20: setup is clear and runnable
- 10-15: mostly acceptable
- 0-9: major running issues

## 2. Protocol understanding and implementation, 30

Check:
- Whether the required protocol is basically followed
- Whether request handling matches the task expectations

Suggested scoring:
- 24-30: protocol handling is mostly correct
- 15-23: partial completion
- 0-14: major protocol issues

## 3. TCP communication ability, 20

Check:
- Whether TCP communication is established
- Whether send/receive flow works basically correctly

Suggested scoring:
- 16-20: communication flow works well
- 10-15: partially works
- 0-9: major issues

## 4. Test results and correctness, 20

Check:
- Whether core tests pass
- Whether core behavior is correct

Suggested scoring:
- 16-20: core tests/behavior mostly correct
- 10-15: partial completion
- 0-9: major issues

## 5. Code quality and README, 10

Check:
- Whether code is basically readable
- Whether README explains how to run it

Suggested scoring:
- 8-10: readable and explained
- 5-7: acceptable
- 0-4: messy or undocumented
